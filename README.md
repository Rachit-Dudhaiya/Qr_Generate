# Qr_Generate

A simple web app to generate QR codes from text or URLs.

## Features
- Enter text or a URL
- Generate a QR code instantly
- Download the QR code image

## Files
- `index.html` — app structure
- `style.css` — app styling
- `script.js` — QR code generation logic

## Usage
1. Open `index.html` in your browser.
2. Enter the text or URL you want to encode.
3. Click the Generate button.
4. Save or scan the generated QR code.
