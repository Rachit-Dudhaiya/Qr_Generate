# Qr_Generate

A simple web app to generate and scan QR codes.

## Features
- Enter text or a URL to generate a QR code
- Download the generated QR image
- Scan QR codes using the mobile camera
- Open detected links directly from the scanner

## Files
- `index.html` — app structure and scanner UI
- `style.css` — app styling and responsive layout
- `script.js` — QR generation and scanning logic

## Usage
1. Open `index.html` in your browser.
2. Use the Generate tab to create a QR from text or a URL.
3. Download the QR image if needed.
4. Switch to the Scan tab to start the camera and scan a QR code.
5. Tap Open link when a URL is detected.

## Notes
- Scanner works best on HTTPS or localhost.
- Mobile vibration is triggered on successful QR detection when supported.
